# Security Policy

## Supported Versions
Only the latest version of this project is currently supported with security updates.

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please **do not open a public issue.** Instead, please report it via the **"Security" tab** on GitHub using the **"Private vulnerability reporting"** feature. Alternatively, you can contact the maintainer directly if an email is provided in the profile.

I will acknowledge your report within 48 hours and provide a timeline for a fix.
